# ubuntu-dev-setup-especially-for-cn

本项目为 ubuntu 个人用开发机器的配置, 适合图形化界面使用.

参考项目(Reference): [ubuntu-dev-setup](https://github.com/lukemorales/ubuntu-dev-setup)

测试环境: Ubuntu 20.04.1 x86_64

## 软件列表

- git
- python3 (pip)
- clash
- nvm
- node
- yarn
- vscode
- docker
- docker-compose
- mongodb
- redis
- zsh (oh-my-zsh)
- dbeaver
